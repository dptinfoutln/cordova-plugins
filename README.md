# Cordova-Plugins

University Of Toulon Master DAMP

2017-2018 Hybride Development Custom

Alert Plugin & Google Maps

## Requirements

- Cordova 7.1.0

## Installation

### Plugin

- `cd android-plugin/app`
- `cordova platform add android@[version]`
- `cordova plugin add ../plugin`
- `cordova run`
